# 模板消息

`Baas.sendTemplateMessage(type, value)`

##### 参数说明

|   参数名   |   类型   |  必填  |   描述   |
| :-----------: | :----: | :--: | :------------------------ |
| type | String | 是 | 'form_id' or 'prepay_id' |
| value | String |  是   | form_id or prepay_id |

### 注意事项

在进行以上操作时，请确保 sdk 版本在 v1.1.0 或以上
